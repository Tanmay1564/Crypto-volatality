# Low Level Design (LLD)

- src/data_preprocessing.py: load and clean CSV
- src/features.py: functions to add engineered features
- src/train.py: end-to-end training and model saving
- src/predict.py: load model and predict on new data
- deployment/streamlit_app.py: demo UI for predictions
- models/: store trained models (rf_baseline_model.pkl)
